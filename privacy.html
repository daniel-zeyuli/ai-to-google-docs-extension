Privacy Policy

AI Chat Exporter for Google Docs respects your privacy.

Information We Process

The extension may process:

  • Website content from AI chat platforms (ChatGPT, Gemini, Claude) that the user chooses to export.
  • Authentication information required to connect to the user's Google account through OAuth.

How We Use Information

The data is used only to:

  • Read AI chat responses that the user chooses to export
  • Create a Google Docs document in the user's Google Drive
  • Preserve formatting including supported math equations

Data Storage & Security

This extension does not store user data on external servers. All processing happens locally in the browser.

The extension uses the Chrome Identity API to authenticate users with their Google account. The OAuth token is stored locally within your browser and is never transmitted to any developer-owned or third-party servers.

Google API Limited Use Disclosure

AI Chat Exporter for Google Docs's use and transfer to any other app of information received from Google APIs will adhere to the Google API Services User Data Policy, including the Limited Use requirements.

Data Sharing

No user data is sold, rented, or shared with third parties.

Data may be transmitted only to Google services when the user authorizes the extension to create a document in Google Drive.

Changes

This policy may be updated if the extension functionality changes.

Contact

If you have questions, contact the developer.
