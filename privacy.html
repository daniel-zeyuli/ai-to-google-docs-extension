Privacy Policy

Last updated: May 2, 2026

Save ChatGPT, Claude & Gemini to Google Docs respects your privacy. This policy explains what information the extension processes, why, and what we do (and don't do) with it.

Information We Process

The extension processes the following data, all of it locally in your browser:

  • AI chat content — text, images, code blocks, tables, and math equations from ChatGPT, Gemini, and Claude responses that you explicitly choose to export.
  • Google account authentication — an OAuth access token issued by Google when you sign in. This token allows the extension to create and update documents in your Google Drive on your behalf. The extension requests only the drive.file scope, which restricts access to files the extension itself creates — it cannot read or list any other files in your Drive.
  • User preferences — your default export mode (single response, full conversation, or selection panel), preferred export destination (Google Drive, local .docx, or Markdown .md), destination folder, and a record of the last Google Doc associated with each AI conversation. Stored locally using Chrome's storage API.
  • Image data — when you export a response containing images, the extension fetches the image from its source URL (typically a CDN operated by the AI platform), converts it to PNG, and embeds it into the exported document. No image data is stored beyond the export operation.

How We Use Information

Information is used solely to provide the export functionality:

  • Read the AI chat response you choose to export.
  • Create a new Google Doc in your Google Drive (organized into per-platform subfolders: AI Chat Exports / ChatGPT, / Gemini, / Claude), or append content to an existing one.
  • Alternatively, download the export as a local .docx or .md file directly to your device — no Google account required for these modes.
  • Preserve formatting including headings, lists, tables, code blocks, math equations (rendered natively in Google Docs), and embedded images.
  • Include an MLA-style citation header so exported documents are self-contained and citable.
  • Remember your preferences so the extension behaves consistently between sessions.

Permissions Explained

The extension requests the following permissions:

  • identity — required for Google OAuth sign-in (only used when exporting to Google Drive).
  • storage — required to remember your preferences locally in your browser.
  • downloads — required to save exported files to your device when using the local .docx or Markdown .md destinations.
  • Access to chatgpt.com, claude.ai, gemini.google.com — required to read the AI response you choose to export.
  • Access to googleapis.com and docs.googleapis.com — required to upload documents to Google Drive and to append text to existing Google Docs (used only when Drive is the selected destination).
  • Optional access to image CDN domains — requested only at the moment you export a response containing an image. The extension needs to fetch the image from whichever CDN domain serves it (for example, OpenAI's oaiusercontent.com or Google's googleusercontent.com). This permission is requested per origin via Chrome's native permission dialog and is never requested on install. If you decline, the export continues normally and images are replaced with an [Image] placeholder.

Data Storage & Security

The extension does not store user data on any external server. All processing — including markdown parsing, document generation, and image conversion — happens entirely in your browser.

The OAuth token issued by Google is stored locally in your browser via Chrome's secure identity API and is never transmitted to any developer-owned or third-party server. You can revoke the extension's Google access at any time by visiting https://myaccount.google.com/permissions.

User preferences and conversation-to-document mappings are stored locally via Chrome's storage API and remain on your device. They are not synced to any remote server by the extension.

Google API Limited Use Disclosure

Save ChatGPT, Claude & Gemini to Google Docs's use and transfer to any other app of information received from Google APIs will adhere to the Google API Services User Data Policy, including the Limited Use requirements.

Specifically, the extension uses only the drive.file OAuth scope — the narrowest available scope — which limits Drive API access exclusively to files the extension itself creates. Data obtained through the Google Drive API and Google Docs API is used solely for the user-facing feature of creating and updating documents the user explicitly exports. It is not used for advertising, transferred to third parties, or accessed by humans, except as required by law.

Data Sharing

No user data is sold, rented, or shared with third parties.

Data is transmitted only to:

  • Google services — when you authorize the extension to create or update a document in your Google Drive.
  • Image CDN domains — when you export a response containing images and grant permission to fetch from that origin. Only the image URL is requested; no other data is sent.

Analytics & Tracking

The extension does not include any analytics, telemetry, error reporting, or usage tracking. No data about how you use the extension leaves your browser.

Changes
